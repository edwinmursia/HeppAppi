# HeppaAppi
This is a mobile application which is developed for booking horse riding times from an instructor. The app is currently only in Finnish.

## The app:
### 1. This is the landing screen.
![2FD39CCF-B7AF-4FB8-9850-5D74A0893759](https://user-images.githubusercontent.com/59910760/146366156-f49b054a-6ccd-4ef0-b627-8d20f5a53669.png)

### 2. From the landing screen you can choose to register to the app or sign in with an existing profile.
![930719FD-F5C8-4221-B6D3-88A64973BFE7](https://user-images.githubusercontent.com/59910760/146366349-1f468ef6-b6f2-4813-a2c9-5e627abf255f.png)

### 3. This is the main screen of the app where the calendar is displayed. From here the user can inspect and book all free horse riding times from the calendar which are marked with a green background.
![B15F0DBE-AC58-4EE3-BD68-D3FB485C07FF](https://user-images.githubusercontent.com/59910760/146367522-aec528a1-d007-41e9-b81f-000bdcef96f5.png)

### 4. When the user has booked a riding time it will be displayed in the main screen in a grey reservation component. The reservation can be modified or deleted.
